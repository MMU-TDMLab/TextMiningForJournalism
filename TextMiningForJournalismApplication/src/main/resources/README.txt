This application is part of the OpenMinTeD tender project 'Text Mining for 
Journalism'. It comprises of a set of UIMA components which wrap the Stanford
NLP Tagger and extract People, Location and Time phrases. These are ported into
a separate journalism type system via a series of separate components, giving 
the opportunity for future adaptations to the system.

For further details please contact m.shardlow@mmu.ac.uk
